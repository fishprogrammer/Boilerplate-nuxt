نسخه 1.0.2 — Soft Store فاز ۱

  • جدید صفحات عمومی دو زبانه: /fa/، /en/، فروشگاه، جزئیات محصول، دسته‌بندی
  • جدید وبلاگ عمومی: /fa|en/blog
  • جدید Mockها و composableها: useCatalog، useCommerce، useLicensing، useApi، useSeoFromApi
  • جدید SEO: robots.txt، sitemap stub، JSON-LD، canonical و hreflang
  • جدید مسیرهای /panel/** و /admin/** به پنل فعلی redirect می‌شوند
  • جدید صفحه /panel/checkout با اعتبارسنجی کوپن (mock)
  • جدید env: NUXT_PUBLIC_CATALOG_API_LIVE و فلگ‌های commerce/licensing
  • تغییر routeRules برای zoneهای عمومی و panel/admin

نسخه 1.0.2 — Deploy و Liara

  • رفع تزریق entry.*.css در HTML تولیدشده (استایل production)
  • رفع بیلد Windows/Liara با مسیر مستقیم nuxt.mjs
  • رفع PWA Workbox: حذف precache فایل‌های HTML (Mixed Content روی HTTPS)
  • رفع خطای dev rollupOptions.input با ssr: false
  • تغییر Tailwind و PWA در dependencies برای نصب روی Liara
  • رفع TypeScript برای @tailwindcss/vite
